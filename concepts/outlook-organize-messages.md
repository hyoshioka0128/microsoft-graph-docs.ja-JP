<span data-ttu-id="86613-113">既定以外のカスタム フォルダーの場合、そのフォルダー パスがわかっている場合は、まず `/users/{id}/mailfolders` ショートカットを使用してルート レベルを取得したり最上位レベルのすべてのフォルダーを取得したりして、フォルダーにアクセスできます:</span><span class="sxs-lookup"><span data-stu-id="86613-113">For a custom, non-default folder, if you know its folder path, you can access the folder by first using the `/users/{id}/mailfolders` shortcut to get to the root level and get all the top level folders:</span></span>
