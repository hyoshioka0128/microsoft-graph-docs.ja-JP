# <a name="installintent-enum-type"></a>installIntent 列挙型

> **注:** Intune のコントロールおよびポリシーの構成に Microsoft Graph API を使用するには、これまでどおりに顧客が Intune サービスの[適切なライセンス](https://go.microsoft.com/fwlink/?linkid=839381)を持っている必要があります。

管理者が選択したインストールのインテントに使用できる値
## <a name="members"></a>メンバー
|メンバー|値|説明|
|:---|:---|:---|
|利用可能|0|インストールのインテントが利用できます。|
|必須|1|インストールのインテントが必要です。|
|アンインストール|2|インストールのインテントをアンインストールします。|
|availableWithoutEnrollment|3|インストールのインテントを登録しなくても利用できます。|
