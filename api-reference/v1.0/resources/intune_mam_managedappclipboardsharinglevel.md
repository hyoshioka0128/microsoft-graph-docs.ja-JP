# <a name="managedappclipboardsharinglevel-enum-type"></a>managedAppClipboardSharingLevel 列挙型

> **注:** Intune のコントロールおよびポリシーの構成に Microsoft Graph API を使用するには、これまでどおりに顧客が Intune サービスの[適切なライセンス](https://go.microsoft.com/fwlink/?linkid=839381)を持っている必要があります。

アプリケーションとの間でデバイスのクリップボードを共有する可能性があるレベルを表します
## <a name="members"></a>メンバー
|メンバー|値|説明|
|:---|:---|:---|
|allApps|0|共有はすべてのアプリケーション間で管理するしないかを許可されます|
|managedAppsWithPasteIn|1|共有は、貼り付けが有効になっている管理されるすべてのアプリケーションの間で許可されます。|
|managedApps|2|共有はすべての管理されるアプリケーションの間で許可されます|
|ブロックされています|3|アプリケーション間の共有が無効になっています|
