# <a name="vpptokensyncstatus-enum-type"></a>vppTokenSyncStatus 列挙型

> **注:** Intune のコントロールおよびポリシーの構成に Microsoft Graph API を使用するには、これまでと同様に顧客が Intune サービスの[適切なライセンス](https://go.microsoft.com/fwlink/?linkid=839381)を所持している必要があります。

Apple Volume Purchase Program トークンに関連付けられている可能性のある同期の状態。
## <a name="members"></a>メンバー
|メンバー|値|説明|
|:---|:---|:---|
|なし|0|既定の状態。|
|InProgress|1|最新の同期が進行中。|
|completed|2|前回の同期は正常に完了。|
|failed|3|前回の同期は失敗。|
