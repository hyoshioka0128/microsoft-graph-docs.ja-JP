Exchange コネクタの種類です。
## <a name="members"></a>メンバー
|メンバー|値|説明|
|:---|:---|:---|
|onPremises|0|オンプレミスの Exchange 環境に接続します。|
|ホスト|1|O365 マルチ テナント型の Exchange 環境に接続します。|
|serviceToService|2|O365 マルチ テナント型の Exchange 環境に直接 Intune サービスに接続します。|
|専用|3|O365 専用の Exchange 環境に接続します。|
