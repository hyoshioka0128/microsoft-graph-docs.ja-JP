フランスのテレビのコンテンツの規制ラベル
## <a name="members"></a>メンバー
|メンバー|値|説明|
|:---|:---|:---|
|allAllowed|0|既定値、すべてのテレビ番組コンテンツを許可します。|

|agesAbove12|3|-12 分類が 12 未満の子供にお勧めできません。|
|agesAbove16|4|-16 のクラス分けは 16 の子にはお勧めできません。|
|agesAbove18|5|-18 の分類は 18 才未満の方のためお勧めできません。|
