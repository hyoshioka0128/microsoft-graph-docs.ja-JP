# <a name="windowsdevicetype-enum-type"></a>windowsDeviceType 列挙型

> **注:** Microsoft graph API for Intune では、テナントに対して[アクティブな intune ライセンス](https://go.microsoft.com/fwlink/?linkid=839381)が必要です。

Windows デバイスの種類のプロパティが含まれています。

## <a name="members"></a>メンバー
|メンバー|値|説明|
|:---|:---|:---|
|なし|.0|フラグは設定しません。|
|コンピューター|1 |デスクトップの Windows デバイスの種類がサポートされているかどうか。|
|体|2 |モバイル Windows デバイスの種類がサポートされているかどうか。|
|holographic|4 |Holographic Windows デバイスの種類がサポートされているかどうか。|
|team|8 |チーム Windows デバイスの種類がサポートされているかどうか。|
