Exchange コネクタの同期要求の種類。
## <a name="members"></a>メンバー
|メンバー|値|説明|
|:---|:---|:---|
|fullSync|0|Exchange のすべてのデバイスを検出します。|
|deltaSync|1|デルタ同期ウィンドウの中に更新する Exchange のデバイスのみを検出します。|
