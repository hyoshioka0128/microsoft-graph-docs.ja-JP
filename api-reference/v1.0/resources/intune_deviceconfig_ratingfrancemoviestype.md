フランスの映画評価ラベル
## <a name="members"></a>メンバー
|メンバー|値|説明|
|:---|:---|:---|
|allAllowed|0|既定値、すべての映画コンテンツを許可。|
|allBlocked|1|すべての映画コンテンツを不許可。|
|agesAbove10|2|等級 10 では、10歳以下の児童への映画の上映を禁止。|
|agesAbove12|3|等級 12 では、12 歳以下の児童への映画の上映を禁止。|
|agesAbove16|4|等級 16 では、16歳以下の児童への映画の上映を禁止。|
|agesAbove18|5|等級 18 では、18 歳以下の未成年者への映画の上映を禁止。|
