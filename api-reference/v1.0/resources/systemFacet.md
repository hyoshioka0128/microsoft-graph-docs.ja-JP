# <a name="system-facet"></a>System ファセット

**System** ファセットは、オブジェクトが自身の操作のためにシステムによって管理されていることを示します。
アプリの多くは System ファセットを持つアイテムを無視すべきです。

**注**:現在、このファセットは空ですが、将来のリビジョンでは、このファセットに追加のプロパティが設定される可能性があります。
