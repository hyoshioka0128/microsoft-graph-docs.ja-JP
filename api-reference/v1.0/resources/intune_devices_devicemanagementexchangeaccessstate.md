# <a name="devicemanagementexchangeaccessstate-enum-type"></a>deviceManagementExchangeAccessState 列挙型

> **注:** Intune のコントロールおよびポリシーの構成に Microsoft Graph API を使用するには、これまでどおりに顧客が Intune サービスの[適切なライセンス](https://go.microsoft.com/fwlink/?linkid=839381)を持っている必要があります。

デバイスExchange のアクセスの状態
## <a name="members"></a>メンバー
|メンバー|値|説明|
|:---|:---|:---|
|なし|0|Exchange からはアクセスの状態が検出されませんでした。|
|不明|1|Exchange へのデバイスのアクセス状態は不明です。|
|許可されています|2|デバイスが Exchange へのアクセス権を持っています|
|ブロック|3|デバイスが Exchange内でブロックされています。|
|隔離されています|4|デバイスが Exchangeで隔離されています|
