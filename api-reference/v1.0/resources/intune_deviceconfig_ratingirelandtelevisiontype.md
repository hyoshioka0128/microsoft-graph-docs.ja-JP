アイルランドのテレビ コンテンツの規制ラベル
## <a name="members"></a>メンバー
|メンバー|値|説明|
|:---|:---|:---|
|allAllowed|0|既定値、すべてのテレビ番組コンテンツを許可します|
|allBlocked|1|すべてのテレビ番組コンテンツを許可しません|
|一般|2|GA 区分はすべての対象ユーザーに適しています|
|子供|3|CH 区分は子供に適しています|
|youngAdults|4|YA 区分は 13 歳以上の子供に適しています|
|parentalSupervision|5|PS 区分は、子供のアクセスを制限することについての保護者による考慮を推奨します|
|成人|6|MA 区分は成人に適しています|
