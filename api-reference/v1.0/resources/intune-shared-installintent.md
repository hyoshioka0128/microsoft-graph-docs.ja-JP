管理者が選択したインストールの目的に使用できる値
## <a name="members"></a>メンバー
|メンバー|値|説明|
|:---|:---|:---|
|使用可能|0|使用可能なインストールの目的です。|
|必須|1|目的のインストールが必要です。|
|アンインストール|2|目的のインストールをアンインストールします。|
|availableWithoutEnrollment|3|インストールの目的を登録しなくても使用できます。|
