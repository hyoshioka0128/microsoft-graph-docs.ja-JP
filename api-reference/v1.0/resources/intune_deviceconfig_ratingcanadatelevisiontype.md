カナダのテレビのコンテンツの規制ラベル
## <a name="members"></a>メンバー
|メンバー|値|説明|
|:---|:---|:---|
|allAllowed|0|既定値、すべてのテレビ番組コンテンツを許可|

|parentalGuidance|5|PG、保護者による助言と指導が必要です|
|agesAbove14|6|14+ 区分は 14 歳以上の視聴者が対象です|
|agesAbove18|7|18+ 区分は 18 歳 以上の視聴者が対象です|
