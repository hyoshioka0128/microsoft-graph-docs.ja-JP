## <a name="properties"></a>プロパティ
|プロパティ|型|説明|
|:---|:---|:---|
|scheduledInstallDay|文字列型 (String)|スケジュールされたインストール日 (曜日)。可能な値: `userDefined`、`everyday`、`sunday`、`monday`、`tuesday`、`wednesday`、`thursday`、`friday`、`saturday`。|
|scheduledInstallTime|TimeOfDay|スケジュールされたインストール時刻|
