> **注:** Microsoft graph API for Intune では、テナントに対して[アクティブな intune ライセンス](https://go.microsoft.com/fwlink/?linkid=839381)が必要です。

カナダでの映画の定格ラベル

## <a name="members"></a>メンバー
|メンバー|値|説明|
|:---|:---|:---|
|allallowed|.0|既定値。すべてのムービーコンテンツを許可する|
|allblocked|1 |任意の映画コンテンツを許可しない|
|元帳|2 |G 分類は、すべての年齢に適しています。|
|parentalguidance|3 |PG 分類は、保護者によるガイダンスをアドバイスします。|
|agesAbove14|4 |14A の分類は、14才以上の閲覧者に適しています。|
|agesAbove18|5 |18a 分類は18またはそれ以前の閲覧者に適しています。|
|しか|6 |R 分類は18年以上に制限されています。|
