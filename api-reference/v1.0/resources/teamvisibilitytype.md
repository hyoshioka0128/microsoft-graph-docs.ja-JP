#<a name="teamvisibilitytype-enum-type"></a>teamVisibilityType 列挙型



[チーム](../resources/team.md)の可視性を説明します。 

## <a name="members"></a>メンバー

| メンバー | 値| 説明 |
|:---------------|:--------|:----------|
|プライベート|0|すべてのユーザー、チームを参照してくださいが、所有者のみがチームにユーザーを追加できます。|
|public|1|チームでだれでも参加できます。|
