# <a name="timezonebase-resource-type"></a>timeZoneBase リソースの種類

タイム ゾーンの基本的な表現です。


## <a name="properties"></a>プロパティ
| プロパティ     | 型   |説明|
|:---------------|:--------|:----------|
| name | string | タイム ゾーンの名前。 標準的なタイム ゾーンの名前 ("ハワイ アリューシャン標準時" など) を使用することも、カスタム タイム ゾーンとして "カスタム タイム ゾーン" という名前を指定することもできます。 |
