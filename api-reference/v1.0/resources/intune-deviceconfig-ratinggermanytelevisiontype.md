ドイツのテレビのコンテンツの規制ラベル
## <a name="members"></a>メンバー
|メンバー|値|説明|
|:---|:---|:---|
|allAllowed|0|既定値、すべてのテレビ番組コンテンツを許可します。|

|agesAbove16|5|Ab 16 Jahren、16 のエージングします。|
|大人|6|Ab 18 Jahren、成人のみ|
