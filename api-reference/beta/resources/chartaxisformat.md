# <a name="chartaxisformat-resource-type"></a>ChartAxisFormat リソースの種類

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

グラフ軸の書式設定プロパティをカプセル化します。


## <a name="methods"></a>メソッド
なし
## <a name="properties"></a>プロパティ
なし
