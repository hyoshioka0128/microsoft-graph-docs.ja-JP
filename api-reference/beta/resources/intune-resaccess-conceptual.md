# <a name="manage-resource-access-certificates-in-intune"></a>Intune でのリソース アクセスの証明書を管理します。

> **重要:** Microsoft Graph のベータ版 (/beta) の API はプレビュー中であるため、変更されることがあります。 実稼働アプリケーションでの、これらの API の使用はサポートされていません。

> **注:** Intune のコントロールおよびポリシーの構成に Microsoft Graph API を使用するには、これまでどおりに顧客が Intune サービスの[適切なライセンス](https://www.microsoft.com/en-us/cloud-platform/microsoft-intune-pricing)を持っている必要があります。

Intune の Microsoft グラフ API を使用して PFX 証明書をインポートすることができます。

次の Graph リソースを使用して、Intune での登録を管理できます。

- [PFX 証明書のユーザー](intune-raimportcerts-userpfxcertificate.md)
- [PFX のユーザーの目的](intune-raimportcerts-userpfxintendedpurpose.md)
- [ユーザー PFX の埋め込みスキーム](intune-raimportcerts-userpfxpaddingscheme.md)
