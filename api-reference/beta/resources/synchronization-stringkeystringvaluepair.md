# <a name="stringkeystringvaluepair-resource-type"></a>stringKeyStringValuePair リソースの種類

## <a name="properties"></a>プロパティ
| プロパティ     | 型   |説明|

|Key|String|キー。|
|value|文字列型 (String)|値。|
