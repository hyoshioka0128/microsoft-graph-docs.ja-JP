# <a name="sitepagedata-resource"></a>sitepagedata リソース

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

**sitepagedata**リソースは[webPart][]のプロパティを表します。 これらのプロパティは web パーツによって異なるため、これは固定プロパティを持たない OpenType です。
