VPN トラフィックの規則が関連付けられているアプリケーションの種類を示します。
## <a name="members"></a>メンバー
|メンバー|値|説明|
|:---|:---|:---|
|none|0|トラフィックの規則は、アプリケーションに関連付けられているではありません。|
