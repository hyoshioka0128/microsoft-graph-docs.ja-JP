## <a name="properties"></a>プロパティ
|プロパティ|種類|説明|
|:---|:---|:---|
|ID|String|デバイス管理スクリプト ユーザー状態のエンティティのキーです。|
|successDeviceCount|Int32|デバイスの数の特定のユーザーに成功します。|
|errorDeviceCount|Int32|特定のユーザー エラー デバイスの数。|
|userPrincipalName|String|特定のユーザーのユーザー プリンシパル名です。|
